# Employee Retention Prediction

## 📌 Project Overview
This project aims to predict whether an employee is likely to leave the company using machine learning techniques. It helps organizations reduce employee attrition and improve workforce planning.

## 🎯 Objective
- Predict employee job change behavior
- Identify key factors affecting employee retention

## 📊 Dataset
HR dataset including:
- Education
- Experience
- Company size
- Training hours
- City development index

## ⚙️ Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Power BI

## 🔍 Process
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Model building using:
  - Logistic Regression
  - Random Forest
  - Gradient Boosting

## 📈 Results
- Best model: Gradient Boosting
- Accuracy: ~77.7%

## 💡 Conclusion
The model helps identify employees at risk of leaving and supports data-driven HR decisions.
